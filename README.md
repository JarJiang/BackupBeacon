## 安全说明

- 数据库连接密码会以加密形式持久化存储。
- 系统支持“一次初始化密钥”：首次进入页面时设置，密钥持久化到 `./data/crypto.key`。
- 也可通过环境变量 `BACKUPBEACON_CRYPTO_KEY` 提前注入（优先级更高）。
- 密钥变更后，历史加密数据将无法解密，请谨慎操作。
